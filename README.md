# web-haptics

Haptic feedback for the web.

## Installation

```shell
pnpm i web-haptics
```

## Usage

```tsx
import { WebHaptics } from "web-haptics/react";

<WebHaptics>Hello world</WebHaptics>;
```

# Contributing

## Install dependencies

```sh
pnpm install:all
```

## Dev/Watch Library and Example

```sh
pnpm dev
```

## Build Library

```sh
pnpm build
```
